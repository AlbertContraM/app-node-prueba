# Simple Nodejs App

## USE

* Node
* Express
* MongoDB
* Docker
* Docker-compose


## Instalation

1. Clone the repository
1. Run $npm install
1. Run $node index.js


## Functions

* http://localhost:3010/     : Send a hello message
* http://localhost:3010/new  : Add a new random user to database
* http://localhost:3010/all  : Get all the current users
